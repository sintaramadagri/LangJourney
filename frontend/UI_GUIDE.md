# LangJourney UI 架构说明

## 页面结构

项目已重构为精美的多页面架构，每个功能都有独立的页面。

### 页面列表

1. **主页（仪表板）** - `/`
   - 展示平台概览
   - 快速操作入口
   - 学习统计数据
   - 平台特性介绍

2. **创建学习路径** - `/create-path`
   - 创建新的学习路径
   - 设置路径信息（名称、描述、语言、难度）
   - 配置任务数量（FHEVM加密）
   - 预计学习时长

3. **提交学习任务** - `/submit-task`
   - 选择路径和任务
   - 上传学习成果
   - 文件附件支持
   - 自评分数（FHEVM加密）

4. **审核任务** - `/verify`
   - 查看待审核列表
   - 在线评分和反馈
   - 加密状态下更新分数
   - 批准/拒绝提交

5. **我的证书** - `/certificates`
   - 查看已获得的证书
   - 下载和分享证书
   - 学习统计概览
   - 链上验证

## UI 特性

### 设计风格
- **现代化**: 使用渐变色、阴影、圆角等现代设计元素
- **移动优先**: 响应式设计，完美适配手机、平板、桌面
- **视觉反馈**: 悬停效果、动画过渡、加载状态
- **色彩体系**:
  - 主色: 蓝色 (#2B6EFF)
  - 成功: 绿色 (#2EC27E)
  - 警告: 黄色
  - 错误: 红色
  - 紫色: 审核功能
  - 橙色: 证书功能

### 核心组件

#### Navigation
- 顶部导航栏
- 响应式菜单
- 钱包连接状态
- 链ID显示
- 移动端友好

#### WalletGuard
- 钱包连接保护
- 未连接时显示连接提示
- 优雅的错误处理

### 交互特性

1. **表单验证**
   - 必填字段标记
   - 实时验证
   - 错误提示

2. **加载状态**
   - 按钮加载动画
   - 禁用状态
   - 进度提示

3. **消息反馈**
   - 成功提示
   - 错误提示
   - 警告信息
   - FHEVM状态

4. **卡片悬停**
   - 阴影增强
   - 轻微上移
   - 平滑过渡

## 技术栈

- **框架**: Next.js 15 + React 19
- **样式**: Tailwind CSS
- **字体**: Inter + Poppins
- **图标**: Unicode Emoji
- **动画**: CSS Transitions

## 使用说明

### 开发
```bash
npm run dev
```

### 生产构建
```bash
npm run build
npm start
```

## 自定义

### 修改主题色
编辑 `tailwind.config.ts`:
```typescript
colors: {
  primary: {
    DEFAULT: "#2B6EFF",
    dark: "#1E4DB8",
    light: "#5A8FFF",
  },
  // ...
}
```

### 添加新页面
1. 在 `app/` 目录创建新文件夹
2. 添加 `page.tsx` 文件
3. 使用 `WalletGuard` 包装需要钱包的页面
4. 在 `Navigation.tsx` 中添加导航项

## 注意事项

- 所有页面都使用 `"use client"` 指令（因为需要钱包交互）
- FHEVM相关操作会显示加密图标 🔒
- 合约地址目前硬编码，实际应从环境变量读取
- 文件上传目前仅为UI演示，需要集成IPFS
